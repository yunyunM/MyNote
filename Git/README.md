# Git

## 初始化

`git init`

```shell
git init
```



## 查看git状态

`git status`

```
git status
```



## 添加跟踪文件

```
git add <file_name>
git add *//把所有的文件都添加进去
```

## 上传到缓冲区

```
git commit -a -m "添加说明"
```



## 上传到github

```
git push
```
